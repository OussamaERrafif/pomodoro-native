# High-Impact Features Worth Adding

## 1. Smart Scheduling (Very Valuable)
- [x] Smart next-task recommendation with momentum-based scoring and one-line reason
- [x] Best focus window computed from session log timestamps (2-hour peak window)
- [x] "Focus next" banner on Tasks screen with Focus button; peak window hint on timer screen
- [x] Session log (`sessionLog`) recorded per completed session for ongoing time-of-day learning

## 2. Calendar Integration
- [ ] Integrate with Google Calendar, Apple Calendar, Outlook Calendar
- [ ] Auto-block focus sessions, sync deadlines, avoid conflicts
- Dramatically increases daily usage

## 3. Streaks & Achievements
- [x] Gamify existing metrics with focus streaks
- [x] Achievements: "First Tomato", "Centurion", "Week Warrior", "Flow State", etc.
- [x] XP system with levels/titles
- Users love visible progress

## 4. Focus Analytics (Power Feature)
- [x] Connect distraction logs and mood data into insights
- [x] Examples: "Wednesday is your most focused day", "High-mood days produce 2×+ more sessions", "Most distractions happen in the afternoon", "Work is your most active category"
- Turns data into behavioral insight

## 5. AI Focus Coach
- [x] Conversational assistant inside the app
- [x] Examples: "Why have I been unproductive this week?", "Plan my evening.", "Help me recover after burnout.", "What should I focus on right now?"
- Coach has full context: streak, score, tasks, mood, distractions, week summary

## 6. Offline Mode
- [ ] Timer, tasks, history, sounds, and cached AI suggestions should work offline
- Users hate productivity apps that fail without internet

## 7. Cross-Device Sync
- [ ] Phone ↔ tablet ↔ web ↔ desktop sync using Firebase, Supabase, or Appwrite
- Almost expected now

# Features You Probably Don't Need Yet

Avoid adding these too early:
- Full team collaboration
- Marketplace/plugins
- Public social feed
- Overly complex project management
- Crypto/NFT/gimmicks

They increase complexity without improving core retention.

# Biggest Missing Piece Right Now

Your app has productivity tools, tracking, and AI assistance. But the strongest missing layer is **intelligent adaptation**.

The next evolution is:
- AI understanding user behavior
- Predicting focus
- Recommending actions
- Adapting the system automatically

That's what would make the app feel truly next-generation instead of "Pomodoro app + AI."
